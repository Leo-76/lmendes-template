## [1.0.0] - 2025-01-01

### Added
- Initial release
- Authentication (login, register, logout)
- Dashboard layout with sidebar
- Profile page (edit name, email, password)
- Blade components: `card`, `alert`, `button`, `badge`, `modal`
- Helpers: `initials()`, `money()`, `template_config()`, `template_menu()`, `template_theme()`
- `php artisan template:install` command
- Publishable config, views, migrations
- PHPUnit feature & unit tests
